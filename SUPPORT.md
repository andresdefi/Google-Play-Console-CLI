# Support

## Getting Help

- **Bug reports**: [Open an issue](https://github.com/andresdefi/Google-Play-Console-CLI/issues/new?template=bug_report.yml)
- **Feature requests**: [Open an issue](https://github.com/andresdefi/Google-Play-Console-CLI/issues/new?template=feature_request.yml)
- **Questions & discussion**: [GitHub Discussions](https://github.com/andresdefi/Google-Play-Console-CLI/discussions)

## Before Opening an Issue

1. Check existing [issues](https://github.com/andresdefi/Google-Play-Console-CLI/issues) and [discussions](https://github.com/andresdefi/Google-Play-Console-CLI/discussions)
2. Include the output of `gpc version`
3. Include the full command you ran and the error output

## Google Play API Issues

Issues with the Google Play Developer API itself (rate limits, unexpected responses, permission errors) should be reported to [Google's issue tracker](https://issuetracker.google.com/). If you're unsure whether the issue is with `gpc` or the API, feel free to open an issue here and we'll help triage.
