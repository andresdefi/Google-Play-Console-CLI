# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in gpc, please report it through [GitHub Security Advisories](https://github.com/andresdefi/Google-Play-Console-CLI/security/advisories/new).

**Please do not open a public issue for security vulnerabilities.**

### What to include

- A description of the vulnerability
- The potential impact
- Steps to reproduce (if possible)
- Suggested fix (if you have one)

### Response timeline

- **Acknowledgment**: within 48 hours
- **Initial assessment**: within 7 days
- **Fix target**: within 30 days for confirmed vulnerabilities

## Scope

This policy covers the `gpc` CLI tool itself. Issues with the Google Play Developer API should be reported to Google directly.

## Supported Versions

Only the latest release is actively supported with security updates.
